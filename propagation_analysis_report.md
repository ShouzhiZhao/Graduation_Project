# 模拟传播数据分析报告

本报告基于 `analyze_comparison.py` 和 `analyze_sim_properties.py` 的运行结果，对模拟的传播数据进行深入分析，并与真实世界的传播数据（以“疯狂动物城2”为例）进行对比。

## 1. 真实数据与模拟数据对比分析

通过 `analyze_comparison.py` 对比了真实社交网络传播数据与 Agent 模拟数据的关键指标。

### 1.1 总体统计指标对比

| 指标 | 真实数据 (Real) | 模拟数据 (Sim) | 说明 |
| :--- | :--- | :--- | :--- |
| **网络规模 (Nodes)** | 412,141 | 1,000 | 模拟网络为 1000 人的缩微模型 |
| **峰值活跃比例 (Max Active Prop)** | 10.77% | 6.29% | 模拟传播的爆发强度略低于真实数据 |
| **最终覆盖人数 (Final Reach)** | 410,717 | ~192.6 | 真实数据几乎覆盖全网，模拟数据覆盖约 19.3% 的用户 |
| **总互动量 (Total Volume)** | 975,025 | ~930.7 | 平均每人产生约 0.93 次互动 |

### 1.2 趋势相关性分析

尽管绝对数值存在量级差异（由于网络规模不同），但两者在随时间演化的趋势上表现出高度一致性：

*   **活跃度相关性 (Correlation Active): 0.5910**
    *   反映了每日/每步活跃用户数的波动趋势。中等程度的相关性表明模拟能够捕捉到爆发和消退的基本过程，但在具体的波动细节上可能存在差异。
*   **累计覆盖人数相关性 (Correlation Viewers): 0.9385**
    *   极高的相关性表明模拟传播的扩散速度和范围扩展模式与真实情况非常接近。
*   **累计互动量相关性 (Correlation View Counts): 0.9820**
    *   接近 1.0 的相关性说明信息总量的累积增长曲线几乎完全吻合。

### 1.3 结论
模拟系统在**宏观演化趋势**上能够很好地复现真实社交网络的传播特征，特别是在累积增长模式上（S形曲线）表现出极高的拟合度。

## 2. 模拟传播动力学特性分析

通过 `analyze_sim_properties.py` 对模拟数据的传播机制进行了动力学拟合分析。

### 2.1 爆发特性 (Peak Analysis)
*   **峰值出现时间**: Step 2
*   **峰值强度**: 约 62.92 个 Agent 同时活跃
*   **特征**: 传播在模拟开始后迅速达到高潮（第 2 步），显示出极强的新闻时效性和爆发性，随后进入衰退期。

### 2.2 增长曲线分析 (S-Curve Growth)

对“累计互动量 (Volume)”和“累计覆盖人数 (Reach)”进行了 Logistic 和 Gompertz 两种生长模型的拟合。

| 指标 | 最佳拟合模型 | R² (拟合优度) | 饱和容量 (K) | 分析 |
| :--- | :--- | :--- | :--- | :--- |
| **累计互动量 (Volume)** | **Gompertz** | **0.9885** | ~1100 | Gompertz 模型优于 Logistic (R²=0.9805)，说明增长是非对称的，前期增长较快，后期饱和缓慢。 |
| **累计覆盖人数 (Reach)** | **Gompertz** | **0.9329** | ~180 | 同样符合 Gompertz 增长模型，最终覆盖网络约 18% 的节点。 |

**结论**: 传播增长符合经典的 Gompertz 生长曲线，这通常用于描述受限资源下的非对称增长过程（如社交网络中的信息扩散），验证了模拟机制的合理性。

### 2.3 衰减特性 (Decay Analysis)

对峰值过后的活跃度衰减过程进行了分析，对比了幂律衰减 (Power Law) 和指数衰减 (Exponential)。

*   **幂律衰减 (Power Law)**: **R² = 0.8977** (alpha = 0.1863)
*   **指数衰减 (Exponential)**: R² = 0.5487 (lambda = 0.0153)

**结论**: 模拟数据的长尾衰减明显更符合**幂律分布 (Power Law)**。
*   这意味着热度不会像指数衰减那样迅速消失，而是具有长尾效应，能够在较长时间内维持一定的低活跃度。
*   这与真实社交网络中话题关注度的衰减规律（通常由人类注意力的重尾分布导致）高度一致。

---

## 3. 总结

本次分析表明，Agent 模拟系统生成的数据具有极高的逼真度：
1.  **趋势复现**: 在宏观累积趋势上与真实数据相关性高达 **0.98**。
2.  **增长规律**: 服从 **Gompertz 模型**，准确反映了信息在社交网络中受限传播的过程。
3.  **衰减规律**: 呈现 **幂律衰减** 特征，成功复现了社交话题的长尾效应。

该模拟系统能够作为研究社交网络舆情传播的有效实验替身。
